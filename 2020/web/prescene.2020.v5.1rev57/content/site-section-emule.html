server list.

You can use almost all the ports
between 1 and 65535.


You can obtain high id if:

1) your ISP gives you a (real) public IP.

2) you set up correctly the port forwarding
in your router settings.

3) you set up correctly your firewall.

4) your ip doesn't end with a 0 (nought):
always low id on servers.

5) there aren't malware/spyware/virus
in your system (can cause low id on kad).

6) you avoid poorly configured servers
(some servers will give only low id
because of this, even if everything
is/was correctly set up).

get a normal connection.

the obfuscation support
and it allows a normal connection.
