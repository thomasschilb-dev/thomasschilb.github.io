Tastenkombinationen für Windows

Strg + Alt + Entf = Taskmanager aufrufen
Alt + Tab = Windows-Programm wechseln
Alt + Shift + Tab = Windows-Programm wechseln (rückwärts)
Alt + F4 = Programm schließen
Strg + Alt + Tab = Programm wechseln, Ansicht der Programme
Strg + F4 = Fenster innerhalb des Programmes schließen
Strg + Esc = Startmenü aufrufen
Strg + Alt + Esc = zu anderen Programm wechseln
Windowstaste + Tab = Windows-Programm wechseln, Übersicht
Windowstaste + M = alle auf der Taskleiste vorhandenen Fenster minimieren
Windowstaste + D = alle Fenster minimieren bzw. diesen Vorgang rückgängig machen
Windowstaste + R = Programm ausführen
Windowstaste + L = Computer sperren
Windowstaste + E = Windows-Explorer öffnen
Windowstaste + F = Windows-Suche öffnen
Windowstaste + P = Präsentationsmodus
Windowstaste + Pause = Systemeigenschaften öffnen
Druck = Bildschirmfoto des ganzen Bildschirms in die Zwischenablage kopieren
Alt + Druck = Bildschirmfoto des aktiven Fensters erstellen und in die Zwischenablage kopieren
