Aus einer Live CD (Knoppix) heraus Grub 2 neu installieren


1. Festplatten Analyse

sudo mount /dev/sda2 /media
sudo mount -o bind /sys /media/sys
sudo mount -o bind /proc /media/proc
sudo cp /proc/mounts /media/etc/mtab


2. In »chroot« Umgebung wechseln

sudo chroot /media /bin/bash


3. Grub2 neu installieren:

grub-mkconfig -o /boot/grub/grub.cfg
grub-install /dev/sda
reboot


4. Grub2 Pakete vollständig neu installieren

aptitude update
aptitude reinstall grub-common grub-pc os-prober # grub-gfxpayload-lists
