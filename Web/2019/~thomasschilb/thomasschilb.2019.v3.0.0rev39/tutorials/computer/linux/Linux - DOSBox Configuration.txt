1. Start dosbox as user Knoppix
2. Edit '~/.dosbox/dosbox-0.74.conf'
   memsize=64
   cputype=pentium_slow
   keyboardlayout=gr129 (for German keyboard layout)
3. [autoexec]
   # Lines in this section will be run at startup.
   # You can put your MOUNT lines here.

   @echo off
   cls
   CONFIG -set dos keyboardlayout gr129
   mount c /media/sda1/dosbox/
   c:
   dir /w
4. Play any DOSgame (in /media/sda1/dosbox/) from now!
