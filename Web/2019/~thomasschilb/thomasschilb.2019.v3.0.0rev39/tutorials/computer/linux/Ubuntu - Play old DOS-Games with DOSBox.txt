Ubuntu - Play old DOS-Games with DOSBox


1. How to install and setup DOSBox in Linux:

sudo apt-get install dosbox


2. Edit configuration file

Edit '~/.dosbox/dosbox-0.74.conf'

   memsize=64
   cputype=pentium_slow
   keyboardlayout=gr129 (for German keyboard layout)


3. Edit Startup in configuration file goto

[autoexec]
# Lines in this section will be run at startup.
# You can put your MOUNT lines here.

@echo off
cls
CONFIG -set dos keyboardlayout gr129
mount c /media/sda1/dosbox/
c:
dir /w


4. Start DOSBox and play any DOS-Game (e.g. in /media/sda1/dosbox/) now!
