testdisk (cgsecurity.org)

1. No Log
2. Proceed
3. Select New Disk Drive
4. Select Intel
5. Select Analyse
6. Select Quick Search
7. Say ‘Y‘ if it asks if the disk was made in Vista/Win7 (even if it was made in XP say ‘yes’) 

parted

select /dev/sdc             - select device
rm 1                        - erase given partition
mklabel gpt                 - set label to gtp
mkpart primary fat32 0G 8G  - create 8gb partition
name 1 since                - set name to partition (requires GPT)
partprobe                   - to update /etc/fstab
