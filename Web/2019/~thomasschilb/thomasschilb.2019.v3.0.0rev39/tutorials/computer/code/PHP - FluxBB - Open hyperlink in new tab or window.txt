FluxBB - Open hyperlink in new tab or window (tested under FluxBB v1.5.10)


The essential file, if you want to do it is "include/parser.php".


In "function handle_url_tag", you should find:

return '<a href="'.$full_url.'">'.$link.'</a>';


Replace it by:

return '<a target="_blank" href="'.$full_url.'">'.$link.'</a>';


This should do the trick for most of the links you want to open in a separate tab or window.
